# NICE 비표준 인증 방식(소켓통신)

## 환경 구성
- Java version : 17
- Spring-boot version 3.0.4

해당 샘플 소스는 소켓 통신(비 표준창)을 통한 인증 방식 샘플 입니다.

Precondition
 - Nice에서 제공하는 라이브러리를 추가(lib 위치 : /lib/MCheckPlus.jar)
 - IntelliJ 및 build.gradle에 외부 라이브러리 설정
 - 소켓 통신을 위한 siteCode 및 패스워드 확인
 - 공인 IP 등록
